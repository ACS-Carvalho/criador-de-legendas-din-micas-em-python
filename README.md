# criador-de-legendas-din-micas-em-python
Este script permite criar legendas dinâmicas através dos modelos Whisper.
Perfeitas para criar vídeos shorts virais.



Requirements:
É necessário ter instalado o ffmpeg, instruções em: https://www.geeksforgeeks.org/how-to-install-ffmpeg-on-windows/

É necessário instalar a biblioteca faster-whisper: pip install faster-whisper
